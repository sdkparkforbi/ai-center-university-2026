## 🚀 GitHub Pages 배포 방법

1. 이 저장소의 **Settings** → **Pages** 이동
2. **Source**에서 `Deploy from a branch` 선택
3. **Branch**에서 `main` / `/ (root)` 선택 후 **Save**
4. 약 1~2분 후 `https://<username>.github.io/aicenteruniv/` 에서 접속 가능
